The WMDM SDK has dependencies on the Core and Internet Development Platform SDKs.  These SDKs can be 
installed from http://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5.

You can build WMDM SDK by running nmake in a Platform SDK build window.

Please send comments or questions to askdmd@microsoft.com.
